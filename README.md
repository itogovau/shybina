# Book Tracker

**Автор:** [шубина варя]


## Описание программы


Book Tracker — это GUI‑приложение для учёта прочитанных книг. Программа позволяет:

* добавлять книги с указанием названия, автора, жанра и количества страниц;
* просматривать все добавленные книги в табличном виде;
* фильтровать записи по жанру и количеству страниц;
* сохранять данные в формате JSON;
* загружать ранее сохранённые данные из JSON‑файла.

Приложение разработано на Python с использованием библиотеки Tkinter для графического интерфейса и модуля json для работы с данными.
